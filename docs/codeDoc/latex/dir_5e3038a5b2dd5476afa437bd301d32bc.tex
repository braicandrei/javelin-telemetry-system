The {\ttfamily -\/i} flag will make the changes to the file. 
\begin{DoxyCode}{0}
\DoxyCodeLine{clang-\/format\ -\/i\ *.cpp\ *.h}

\end{DoxyCode}
 If you prefer to make the changes yourself, running {\ttfamily clang-\/format} without the {\ttfamily -\/i} flag will print out a formatted version of the file. You can save this to a file and diff it against the original to see the changes.

Note that the formatting output by {\ttfamily clang-\/format} is what the automated formatting checker will expect. Any diffs from this formatting will result in a failed build until they are addressed. Using the {\ttfamily -\/i} flag is highly recommended.\hypertarget{README.md_autotoc_md16}{}\doxysubsubsubsection{\texorpdfstring{clang-\/format resources}{clang-\/format resources}}\label{README.md_autotoc_md16}
